# Workflow Skills

Workflow skills orchestrate multiple task skills and atomic services into a complete, reusable content production process.

Use a workflow skill when the user asks for an end-to-end deliverable that needs planning, checkpoints, multiple generated assets, retries, packaging, or reusable run artifacts.

## Available Workflow Skills

- `koubo.md`: default talking-head content workflow. It prepares a presenter image, spoken script, voice reference, talking-head video or clip package, optional cover, and run artifacts.
